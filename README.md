# learn-livewire

## Required
* php 8
* laravel 8

## Assets
* bootstrap 5
* tailwind
* jquery
* popper

## Install
* add env for your local config and database
  [env.example](https://github.com/platformsh-templates/laravel/blob/master/.env.example)
* run this command
```
~ composer install
~ php artisan migrate:fresh --seed
~ php artisan serve
```
